## 🔄 Typical Workflow

### First Time Setup:
1. Read `QUICK_START.md`
2. Install packages: `pip install -r requirements.txt`
3. Copy `.env.example` to `.env`
4. Add your API key to `.env`
5. Run `python verify_setup.py`
6. Run `python 01_hello_world.py`

### Learning:
1. Open a `.py` file
2. Read the comments at the top
3. Read through the code
4. Run the file
5. Modify and experiment
6. Refer to `QUICK_REFERENCE.md` as needed

### Teaching:
1. Review `TEACHING_CHECKLIST.md`
2. Test all examples work
3. Follow session plans
4. Use `QUICK_REFERENCE.md` for Q&A
5. Share all files with students
